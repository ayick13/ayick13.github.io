Kebijakan Privasi

Terakhir diperbarui: 28 Februari 2025

1. Pengenalan

Kami menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi yang Anda bagikan dengan kami. Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat Anda mengunjungi website kami.

2. Data yang Kami Kumpulkan

Kami dapat mengumpulkan informasi berikut saat Anda mengunjungi website kami:

  • Alamat IP: Kami menggunakan layanan pihak ketiga untuk mengumpulkan alamat IP Anda guna keperluan analitik dan keamanan.
  • User Agent: Informasi tentang browser dan perangkat yang Anda gunakan untuk mengakses website kami.

3. Tujuan Pengumpulan Data

Data yang kami kumpulkan digunakan untuk:

  • Meningkatkan pengalaman pengguna di website kami.
  • Menganalisis tren dan kinerja website.
  • Memastikan keamanan website dari serangan atau penyalahgunaan.

4. Dasar Hukum Pengolahan Data

Kami memproses data Anda berdasarkan kepentingan sah (legitimate interest) untuk meningkatkan layanan dan keamanan website kami. Kami tidak akan menggunakan data Anda untuk tujuan lain tanpa persetujuan Anda.

5. Penyimpanan Data

Data yang kami kumpulkan disimpan secara aman dan hanya disimpan selama diperlukan untuk memenuhi tujuan pengumpulan data.

6. Hak Anda

Sesuai dengan GDPR, Anda memiliki hak berikut:

  • Hak Akses: Anda dapat meminta salinan data pribadi yang kami miliki tentang Anda.
  • Hak Perbaikan: Anda dapat meminta perbaikan data yang tidak akurat.
  • Hak Penghapusan: Anda dapat meminta penghapusan data pribadi Anda.
  • Hak Pembatasan Pemrosesan: Anda dapat meminta pembatasan pemrosesan data Anda.
  • Hak Keberatan: Anda dapat menolak pemrosesan data Anda.
Untuk menggunakan hak Anda, silakan hubungi kami melalui informasi kontak di bawah.

7. Berbagi Data

Kami tidak akan membagikan data Anda kepada pihak ketiga kecuali jika diperlukan oleh hukum atau untuk melindungi hak dan keamanan kami.
